## Response Format

All endpoints return JSON with consistent structure:

```json
{
  "video_id": "dQw4w9WgXcQ",
  "language": "English",
  "model": "meta-llama/llama-3.3-70b-instruct:free",
  "result": "...",
  "usage": {
    "prompt_tokens": 2847,
    "completion_tokens": 312,
    "total_tokens": 3159
  },
  "fallback_used": false
}
```

## Error Handling

The API uses standard HTTP status codes:

| Code | Description |
|------|-------------|
| `200` | Success |
| `400` | Bad Request (invalid parameters) |
| `404` | Not Found (video unavailable or transcripts disabled) |
| `422` | Validation Error (invalid request body) |
| `500` | Internal Server Error |
| `503` | Service Unavailable (all AI models rate-limited) |
| `504` | Gateway Timeout |

Error responses include detailed messages:

```json
{
  "detail": "Transcripts are disabled for this video"
}
```

## Rate Limits

- **Transcript endpoints**: No rate limits
- **AI endpoints**: Subject to OpenRouter model limits
- **Search endpoint**: Subject to YouTube Data API quotas

<Tip>
The API automatically falls back to alternative AI models when rate limits are hit.
</Tip>

## Available AI Models

The API supports 8+ free models with automatic fallback:

| Model | Context | Best For |
|-------|---------|----------|
| `meta-llama/llama-3.3-70b-instruct:free` | 128K | General analysis (default) |
| `google/gemini-2.0-flash-exp:free` | 1M | Long videos |
| `deepseek/deepseek-chat-v3.1:free` | 163K | Technical content |
| `qwen/qwen3-coder:free` | 262K | Code-related videos |

## Quick Examples

### Basic transcript extraction

```bash
curl "https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ"
```

### AI summary with custom model

```bash
curl -X POST "https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ/summarize?model=google/gemini-2.0-flash-exp:free" \
  -H "Content-Type: application/json" \
  -d '{}'
```

### Apply Fabric pattern

```bash
curl "https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ/pattern/extract_wisdom"
```

### Search videos

```bash
curl "https://fastapi-youtube-transcript.up.railway.app/search?q=python+tutorial&max_results=5"
```
