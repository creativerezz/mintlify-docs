## Get your first transcript in 3 steps

Start extracting and analyzing YouTube transcripts with AI-powered insights.

### Step 1: Try the API

No authentication required! Start with a simple transcript extraction:

<CodeGroup>
```bash cURL
curl "https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ"
```

```javascript JavaScript
const response = await fetch(
  'https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ'
);
const transcript = await response.json();
console.log(transcript);
```

```python Python
import requests

response = requests.get(
  'https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ'
)
transcript = response.json()
print(transcript)
```
</CodeGroup>

<Check>
You should see a JSON response with the video transcript, including timestamps and text.
</Check>

### Step 2: Get AI analysis

Now let's get an AI-powered summary of the video content:

<CodeGroup>
```bash cURL
curl -X POST "https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ/summarize" \
  -H "Content-Type: application/json" \
  -d '{}'
```

```javascript JavaScript
const response = await fetch(
  'https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ/summarize',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({})
  }
);
const summary = await response.json();
console.log(summary.summary);
```

```python Python
import requests

response = requests.post(
  'https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ/summarize',
  json={}
)
summary = response.json()
print(summary['summary'])
```
</CodeGroup>

<Check>
You'll get a structured summary with key points, insights, and takeaways from the video.
</Check>

### Step 3: Chat about the video

Ask questions about the video content using the chat interface:

<CodeGroup>
```bash cURL
curl -X POST "https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ/chat" \
  -H "Content-Type: application/json" \
  -d '{"message": "What are the main themes in this video?"}'
```

```javascript JavaScript
const response = await fetch(
  'https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ/chat',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      message: "What are the main themes in this video?"
    })
  }
);
const chat = await response.json();
console.log(chat.assistant_response);
```

```python Python
import requests

response = requests.post(
  'https://fastapi-youtube-transcript.up.railway.app/transcript/dQw4w9WgXcQ/chat',
  json={"message": "What are the main themes in this video?"}
)
chat = response.json()
print(chat['assistant_response'])
```
</CodeGroup>

<Check>
The AI will analyze the transcript and provide contextual answers to your questions.
</Check>
